인터넷 vs 웹
인터넷은 전세계 통신망
웹은 인터넷 공간에서 제공하는 서비스

웹2.0 시대
서로 다른 사용자가 함께 새로운 콘텐츠를 창조할 수 있는 시대
ex) 유투브, 위키피디아, 깃허브

웹 동작
웹(web): 요청과 응답 과정

URL: 웹에서 어떤 대상을 구분하는 주소

서버와 클라이언트
서버(제공자): 응답하는 쪽 (백엔드)
클라이언트(사용자): 요청하는 쪽 (프론트엔드)

웹 표준 기술
HTML5(웹 페이지 구성) + CSS3(스타일 적용) + 자바스크립트(사용자 반응 처리 등등)

HTML5 주요 기능
- 멀티미디어 기능
 - 플러그인 도움 없이도 스스로 음악과 동영상을 재생
- 그래픽 기능
 - 2차원 그래픽 구현
 - 3차원 그래픽 구현
- 통신 기능
 - 서버와 실시간으로 쌍방향 통신을 수행
 - 웹에서 실시간 채팅, 다수의 사람과 온라인 게임, 고객 센터 문의-답변 가능
- 장치 접근
- 오프라인 및 저장소
- 시맨틱
 - 검색 엔진 같은 프로그램이 정보를 분석, 자료를 검색 및 처리해서 제공하는 지능형 웹
- css3 스타일시트
- 웹의 성능 극대화 및 통합

HTML5 기본용어
태그와 요소
요소 - HTML 페이지를 구성하는 각 부품(제목, 본문, 이미지 등)
태그 - 요소를 만들 때 사용하는 작성 방법
ex) <h1> Hello HTML5 </h1>
속성 - 태그에 추가 정보를 부여할 때 사용하는 것
ex) <h1 title="header">Hello HTML5</h1>
주석 - 코드 설명 기록 ( <!-- 주석 --> )

HTML5 페이지의 구조
<!DOCTYPE html>   --> html이라고 명시하는것
<html>
<head>
	<title>Hello HTML5</title>  -> 웹 브라우저에 표시하는 제목을 지정
</head>
<body>	-> 사용자에게 실제로 보이는 부분을 작성하는 곳
</body>
</html>

html 태그내부에 넣을 수 있는 태그
lang, meta, title, script, link, style, base

스타일 시트 작성과 실행
- 내부 스타일
 - 스타일 시트 짧은 경우
- 외부 스타일 <link rel="stylesheet' href="~.css">
 - 프로젝트 규모가 클 경우
자바스크립트도 같음
- 내부
- 외부 <script src="~.js"></script>

글자 태그
제목 글자
h1 첫 번쨰로 큰 제목 글자 생성
h2 두 번쨰로 큰 제목 글자 생성
h3 세 번쨰로 큰 제목 글자 생성
...

본문 글자
p 본문 문단 생성
br 줄 바꿈
hr 수평 줄 삽입

앵커태그
하이퍼텍스트: 사용자의 선택에 따라 특정 정보로 이동하는 조직된 문서
a 태그: 다른 웹 페이지나 웹 페이지 내부의 특정 위치로 이동
<a href="http://hanbit.co.kr">한빛 미디어</a>
1 절대 경로
2 상대 경로
3 아이디 경로
4 메일 경로

글자 모양 태그
b 굵은 글자
i 기울어진 글자
small 작은 글자
sub 아래첨자
sup 위 첨자
ins 밑줄 글자
del 취소선이 그어진 글자

목록 태그
ul 순서가 없는 목록
ol 순서가 있는 목록
li 목록 요소 생성

테이블 태그
table 표 삽입
tr 표에 행 삽입
th 표의 제목 셀 생성
td 표의 일반 셀 생성

테이블 태그의 속성
table - border 표의 테두리 두께 지정
th, td - colspan 셀의 너비 지정, rows 셀의 높이 지정

## 주요 규칙
- 변수명과 함수명은 다르게 사용
- 소스 파일 하나는 200줄 이내로 작성
- 한 줄의 길이는 800자 이내로 작성
- 함수의 내용은 70줄 이내로 작성
- 하나의 문장을 2줄로 작성하는 경우
- 수준이 동일한 문장은 시작 위치를 동일하게
- 주석의 내용
